# Axis Model

## Axis 1 — Exploration vs Safety-Control

Measures whether the person moves toward the unknown or reduces exposure.

| Side | Tags |
|---|---|
| Exploration | `EXP`, `RISK`, `ACT`, part of `INT` |
| Safety-Control | `SAF`, `CTRL`, `OBS`, `AVD`, `WAIT` |

Possible outputs:

- High Exploration
- Controlled Exploration
- Safety-Dominant
- Observation Before Action
- Avoidant Safety

## Axis 2 — Thinking Style

Measures how the person processes reality.

| Style | Tags |
|---|---|
| Analytical | `ANA`, `OBS`, `CTRL` |
| Intuitive/Symbolic | `INT`, `MEAN`, `OBS` |
| Practical | `PRAC`, `SAF`, `CTRL` |
| Social/Relational | `SOC`, `COOP`, `REC` |
| Action-Based | `ACT`, `RISK`, `LEAD` |

## Axis 3 — Relationship and Distance Pattern

Measures closeness, autonomy, observation, and recognition.

| Pattern | Tags |
|---|---|
| Social closeness | `SOC`, `COOP` |
| Recognition-seeking | `REC`, `SOC` |
| Independence | `IND`, `CALM`, `AVD` |
| Social observation | `OBS`, `ANA`, `SOC` |
| Responsibility toward others | `RESP`, `COOP`, `SOC` |

## Axis 4 — Agency, Leadership, and Control

Measures how the person exerts influence.

| Mode | Tags |
|---|---|
| Direct leadership | `LEAD`, `ACT`, `RISK` |
| Structured control | `CTRL`, `ANA`, `PRAC` |
| Indirect control | `OBS`, `ANA`, `WAIT` |
| Cooperative agency | `COOP`, `RESP`, `SOC` |
| Withdrawal / non-engagement | `AVD`, `IND`, `CALM` |

## Axis 5 — Ambiguity and Fear Response

Measures behavior when information is incomplete.

| Response | Tags |
|---|---|
| Investigate directly | `ACT`, `RISK`, `EXP` |
| Analyze first | `OBS`, `ANA`, `SAF` |
| Freeze / wait | `WAIT`, `SAF`, `ANX` |
| Avoid | `AVD`, `IND`, `SAF` |
| Protect others | `RESP`, `SOC`, `SAF` |

High-weight evidence:

- Q2
- Q7
- Q11
- Q16

## Axis 6 — Deep Motive Structure

Mainly from Q19 and Q20, supported by repeated tags.

| Motive | Evidence |
|---|---|
| Truth motive | Q20A, `MEAN`, `ANA`, `CTRL` |
| Power motive | Q20B, `CTRL`, `LEAD`, `REC` |
| Love/belonging motive | Q20C, `SOC`, `COOP`, `MEAN` |
| Knowledge motive | Q20D, `MEAN`, `ANA`, `CTRL` |
| Security motive | Q19D/B/C, `SAF`, `CTRL`, `OBS` |
| Social-control motive | Q19A, `OBS`, `SOC`, `CTRL` |
